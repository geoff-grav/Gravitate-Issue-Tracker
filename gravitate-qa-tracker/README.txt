== Description ==

Author: Gravitate http://www.gravitatedesign.com

Description: This is Plugin allows you and your users to Track Website issues.  Allows for team and client collaboration.  It will also take screenshots via HTML5 and Javascript.  No need for a client to install any programs.

==Requirements==

- PHP 5.2 or above
- MySQL 5 or above
- WordPress 3.5 or above


==How to Use==

Enable the Plugin in the Plugins Page. The use the default url to access the tracker page "/qatracker"


== Installation ==

1. Upload the `gravitate-qa-tracker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can configure the Settings in `Settings` -> `Gravitate QA Tracker`
